# Clawpack - Multi-Agent AI Ecosystem

## Agents
- eagleclaw: Main coding assistant
- crustyclaw: Bug detection  
- rustypycraw: Code generation
- claw-coder: Python AI
- claw-code: Claude reimplementation
- agentforlaw: Legal contracts
- sysclaw: System maintenance

## Shared Resources
- Shared memory: ~/dev/claw-shared
- Database: ~/.claw_memory/shared_memory.db
- 19 languages at Level 5

## Commands
- Install: ./install.sh
- Run sysclaw: cd ~/dev/sysclaw && python3 sysclaw.py
- Check mastery: sqlite3 ~/.claw_memory/shared_memory.db "SELECT COUNT(*) FROM language_proficiency;"
