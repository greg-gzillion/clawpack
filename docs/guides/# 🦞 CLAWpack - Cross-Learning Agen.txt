# 🦞 CLAWpack - Cross-Learning Agent Workspace
## A Tethered Multi-Agent Legal Research & Knowledge System

---

## 📋 Overview

CLAWpack is a **tethered multi-agent system** where 19 specialized AI agents share knowledge through a common SQLite database. Each agent has a specific domain expertise, but they all learn from each other through **shared memory**.

### Core Philosophy
- **No silos** - Every agent can learn from every other agent
- **Shared memory** - A central database where all knowledge lives
- **Cross-agent search** - Query all agents' knowledge from any agent
- **Free & open** - No paywalls, all data is local

🔗 How Agents Learn From Each Other
1. Writing to Shared Memory
python
# Any agent can save knowledge
def save_to_shared_memory(query, response, agent_name):
    conn = sqlite3.connect(SHARED_DB)
    c = conn.cursor()
    c.execute('''INSERT INTO cross_agent_knowledge 
                 (query, response, agent_source, timestamp)
                 VALUES (?, ?, ?, ?)''',
              (query, response, agent_name, datetime.now()))
    conn.commit()

Adding a New Agent
python
# 1. Create agent directory
# 2. Implement shared memory integration
# 3. Register with unified system
# 4. Start saving knowledge to shared DB

class NewAgent:
    def __init__(self):
        self.SHARED_DB = Path.home() / ".claw_memory" / "shared_memory.db"
    
    def save_knowledge(self, query, answer):
        conn = sqlite3.connect(self.SHARED_DB)
        c = conn.cursor()
        c.execute('''INSERT INTO new_agent_knowledge 
                     (query, response) VALUES (?, ?)''',
                  (query, answer))
        conn.commit()
