## Cirrhosis
| Feature | Description | URL |
|---------|-------------|-----|
| Definition | Diffuse hepatic fibrosis with nodular regeneration | https://www.ncbi.nlm.nih.gov/books/NBK482419/ |
| Etiologies | Viral hepatitis (HBV, HCV), alcohol, NAFLD, autoimmune, biliary, genetic | https://www.ncbi.nlm.nih.gov/books/NBK482419/ |
| Compensated | No complications; Monitor for decompensation, HCC | https://www.aasld.org |
| Decompensated | Ascites, variceal hemorrhage, hepatic encephalopathy, jaundice | https://www.aasld.org |
| Child-Pugh Score | Bilirubin, Albumin, INR, Ascites, Encephalopathy (A=5-6, B=7-9, C=10-15) | https://www.mdcalc.com/child-pugh-score-cirrhosis-mortality |
| MELD-Na Score | Bilirubin, Creatinine, INR, Sodium; Prioritizes transplant allocation | https://optn.transplant.hrsa.gov |

## Metabolic Liver Diseases
| Disease | Pathophysiology | Key Features | Diagnosis | Treatment | URL |
|---------|-----------------|--------------|-----------|-----------|-----|
