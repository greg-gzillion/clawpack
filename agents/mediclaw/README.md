\# 🏥 Mediclaw - Medical Information Agent



Medical agent with homeopathy, first aid, medication info, and cross-learning capabilities.



\## Commands

\- `/ask \[question]` - Medical questions

\- `/homeo \[remedy]` - Homeopathic remedy info

\- `/emergency \[symptoms]` - Emergency symptom checker

\- `/firstaid \[situation]` - First aid guidance



\## Features

\- Evidence-based medical information

\- Homeopathic remedy profiles

\- Medication database

\- Cross-learning with other agents
