agentforlaw - The Ultimate Agent of Law

options:
  -h, --help            show this help message and exit
  --agencies            List regulatory agencies
  --agency AGENCY       Get agency info
  --domains             List law domains
  --remember KEY VALUE  Store in shared memory
  --recall RECALL       Recall from shared memory
  --agents              Show other claw agents
  --statute STATUTE     Look up US Code statute
  --case CASE           Search case law
  --cfr CFR             Look up Code of Federal Regulations
  --constitution        Access Constitution
  --article ARTICLE     Constitution article number
  --section SECTION     Constitution section number
  --amendment AMENDMENT
                        Constitution amendment number
  --draft-contract {service,sale,employment,lease,partnership,loan}
                        Draft a contract
  --draft-will          Draft a last will
  --draft-trust         Draft a living trust
  --draft-estate {power_of_attorney,healthcare_directive,living_will}
                        Draft an estate document
  --parties PARTIES     JSON string of party information
  --provisions PROVISIONS
                        JSON string of provisions
  --list-clauses        List contract clauses
  --clause CLAUSE       Get a contract clause
  --define DEFINE       Define a legal term
  --list-terms          List legal terms
  --analyze ANALYZE     Analyze legal question using AI
  --list-models         List all available LLM models
  --model MODEL         Specify model (e.g., 'llama-3.3-70b-versatile')
                        LLM provider
  --recommend RECOMMEND
                        Get model recommendation for a legal task
