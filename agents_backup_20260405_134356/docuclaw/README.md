\# 📝 Docuclaw - Document Writing Assistant



Document generation, summarization, proofreading, and formatting agent.



\## Commands



| Command | Description | Example |

|---------|-------------|---------|

| `/write \[topic]` | Generate document | `/write AI agents` |

| `/summarize \[text]` | Summarize content | `/summarize Long text here...` |

| `/outline \[topic]` | Create outline | `/outline blockchain` |

| `/proof \[text]` | Proofread text | `/proof This sentence have error.` |

| `/format \[text]` | Format as markdown | `/format My document text` |

| `/save \[title]` | Save document | `/save My Document` |

| `/load \[title]` | Load document | `/load My Document` |

| `/list` | List saved docs | `/list` |

| `/export \[title]` | Export to file | `/export My Document` |



\## Features

\- ✅ Document generation

\- ✅ Text summarization

\- ✅ Outline creation

\- ✅ Grammar/spelling proofreading

\- ✅ Markdown formatting

\- ✅ Save/load to shared memory

\- ✅ Export to markdown files
